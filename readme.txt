プログラムの解説

このプログラムは、出張管理を行うためのWebアプリケーションです。主な機能は以下の通りです：

出張に関するタスクの管理。
お客様名、出張当日の日付、客先最終日、帰着日などの情報の入力。
各タスクの期限の自動計算と表示。
入力されたデータの保存と復元。
入力内容を印刷するためのPDF生成機能。


主なHTML要素と機能

・タイトルとフォームフィールド
タイトルと、出張に関する基本情報（お客様名、出張当日の日付、客先最終日、帰着日）の入力フィールドがあります。

・タスクリスト
出張前、出張時、帰着後に行うべきタスクがリスト形式で表示されています。それぞれのタスクにはチェックボックスがあり、完了状況を記録できます。

・リセットボタン
入力されたデータをすべてクリアするためのボタンです。

・PDF生成ボタン
入力されたデータを印刷するためのボタンです。印刷ダイアログが開き、PDFとして保存することができます。

JavaScript機能

・日付計算関数 (calculateTaskDates)
出張日、客先最終日、帰着日を元に、各タスクの期限を計算し表示します。

・日付フォーマット関数 (formatDate と formatDay)
日付を "YYYY-MM-DD" 形式および曜日の日本語形式でフォーマットします。

・日付強調関数 (highlightDate)
期限が近いタスクを強調表示するための関数です。3日以内の期限の場合は赤、7日以内の場合は黄色で背景色を変更します。

フォームリセット関数 (resetForm)
入力されたデータをすべてクリアし、チェックボックスの選択を解除します。

・データ保存関数 (saveData)
入力されたデータをローカルストレージに保存します。

・データ読み込み関数 (loadData)
ローカルストレージからデータを読み込み、フォームに反映します。

・PDF生成関数 (generatePDF)
印刷ダイアログを開く関数です。
